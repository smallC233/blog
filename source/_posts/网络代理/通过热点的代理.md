# 通过热点的代理

## 电脑端操作

点开热点 WIFI 的详细信息，复制 IPv4 DNS 服务器。

网络和 Internet > 代理 > 使用代理服务器 > （开） && 填入刚刚复制的代理 IP 地址和手机上设置的{端口号} > 保存。



## 手机端操作

1. 打开热点。



2. 代理软件进行设置（已常开）

### clash

设置 > 覆写 > 复合端口（填一个你喜欢的{端口号}) && 允许来自局域网的连接（启用）



### v2ray

设置 > 进阶设置（在最底下） > HTTP 代理端口（填一个你喜欢的{端口号}) && 允许来自局域网的连接（启用）
