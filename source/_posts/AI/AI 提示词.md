# AI 提示词

## 不允许生成过多无关文件

包含在 `openspec` 的规则内了。



## 画数据流图

```
在文档开头用 mermaid 画个数据流图帮助理解。
把节点里的 \n 全部改为 <br/>，并用 ["..."] 包裹包含 -- 等特殊字符的文本。
数据流向以模块为单位，数据流图的箭头上标注传递的数据是什么。
```



## 画项目结构图

```
在文档开头用 mermaid 画个项目结构图帮助理解，项目结构图用于展示各子模块的关系和功能，要求框出各个子模块所属的父模块，减少箭头数量，只展示大模块间的关系，把关系写在箭头上。
```
