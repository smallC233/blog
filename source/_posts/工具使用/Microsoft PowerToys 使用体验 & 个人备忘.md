# Microsoft PowerToys 使用体验 & 个人备忘



Microsoft PowerToys  是微软官方出品的让你的 Windows 变得更好用的工具集。

同一分类下，大体按照使用频率排序

## 系统工具

### 常用

#### PowerToys Run（快速启动）

`windows 开始菜单搜索`极速版，用来打开在开始菜单注册过的应用，搜索很快，使用体验很丝滑。

**快捷键**：

* 激活：`Alt + Space`

除了打开应用外，还额外支持不少功能，在某些简单场景下比打开 python 要方便一点点：

简单的算术运算如 `2^15`，具体支持的运算有：

* `+, -, *, /, %, ^`
* `()`
* `sqrt(x), log(x), ln(x)`
* `sin(x), cos(x), tan(x)`
* 常量 `pi, e`

单位换算：

* `%% 60 km/h to m/s`
* `%% 114514 s in h`
* `%% 0K to C`
* ``%% 1 rad to deg``

哈希生成（种子不填就是 `rand()` ）：

* `# uuid <种子>`
* `# sha256 <种子>`



#### 文本提取器

以截图的方式选取窗口，然后对窗口进行本地 OCR 识别，将识别到的内容拷贝至剪切板。

支持中文识别，支持深色模式下识别，对黑体、宋体等计算机标准字体的识别准确率很高。

背景是黑白以外的奇怪的颜色时准确率会明显下降。

**快捷键**：

* 激活：`Win + Shift + T` (Text）



#### 颜色选取器

系统级**屏幕取色工具**，可以从屏幕上任意位置捕获颜色值，并**以可配置的格式自动复制到剪贴板**。这个功能对设计师、前端开发、UI 配色校准、截屏取色等场景非常实用，可以彻底替代以前那种截图 → PS 取色的繁琐流程。

**快捷键**：

* 激活：`Win + Shift + C` (Color）

进入取色模式后：

- **左键单击**：取色、**打开编辑器**（保存历史 + 可调色），复制到剪贴板。
- **滚轮滚动**：打开临时放大镜方便精确的取到颜色。
- **右键单击**：取消/关闭，不复制。



#### ZoomIt (屏幕演示辅助)

源自 Windows Sysinternals 的经典工具，专为技术演示和教学场景设计的**屏幕放大与实时标注实用程序**。

**快捷键**：

1. **激活模式**
   - **静止放大 (Static Zoom)**：Ctrl + 1
     - 特性：屏幕画面冻结，鼠标移动仅用于控制视野。点击左键后进入**标注模式**。
     - 退出：按 Esc 或 右键 或 再次按 Ctrl + 1。
   - **仅标注 (Draw without Zoom)**：Ctrl + 2
     - 特性：不放大屏幕，直接在当前画面上进行绘制。
     - 退出：按 Esc 。
   - **实时放大 (LiveZoom)**：Ctrl + 4
     - 特性：屏幕保持活动状态，可继续进行打字、点击菜单等操作，适合实时代码演示。
     - 退出：再次按 Ctrl + 4。
2. 标注与绘图指令 (在标注模式下生效)
   - **画笔颜色**：
     - R (Red 红色) | G (Green 绿色) | B (Blue 蓝色) | Y (Yellow 黄色) | O (Orange 橙色) | P (Pink 粉色)
   - **画笔粗细**：
     - Left Ctrl + 滚轮 上下滚动。
   - **绘图形状**：
     - **直线**：按住 Shift 拖动。
     - **矩形**：按住 Ctrl 拖动。
     - **椭圆**：按住 Tab 拖动。（被我电脑的 `mykeymap` 吃了）
     - **箭头**：按住 Ctrl + Shift 拖动。
   - **其他工具**：
     - **文本输入**：按 T (仅支持英文输入，输入完毕按 Esc 退出文本模式)。
     - **白板/黑板**：按 W (White) 或 K (Black) 切换全屏纯色背景。
     - **清屏**：按 E (Erase)。



### 不常用

#### Awake（保持唤醒）

“软件层面的防休眠工具”，用于在**不改系统电源计划**的前提下，临时或长期阻止 Windows 进入休眠/睡眠/熄屏状态。

暂时用不到。



#### 高级粘贴

对剪贴板内容进行二次处理后再粘贴，比如自动去除格式、转换为纯文本或按指定格式输出。

useless，感觉原生的粘贴和剪切板功能就够用了。

**快捷键**：

* 激活：`win + shift + v`



#### Light Switch（浅色/深色切换）

一键切换 Windows 的浅色 / 深色模式，适合在不同环境或时间段快速调整系统观感，而不必反复进入系统设置。

对我 useless，我有 Auto Dark Mode 自动管理了。



#### 屏幕标尺

**屏幕像素测量工具**，可以在桌面上快速测量任意区域的像素宽度、高度、间距等数值，非常适合设计师、前端开发、产品 UI 校验等需要精确尺寸对齐的场景。

对我 useless 。



#### 快捷键指南

**Windows 键盘快捷键总览弹层**，按下后会在屏幕上显示当前状态下常用的 Win 键组合及其操作提示，帮助你快速学习和掌握系统快捷操作，而不必查手册或记忆一堆组合键。

对我 useless 。



## 窗口和布局

### 常用

#### Always On Top (窗口置顶)

允许用户通过快捷键将任意应用程序窗口锁定在所有其他窗口的顶层（最前端），防止其被遮挡。

建议调低默认厚度，默认有点太厚了。

**快捷键**：

* 置顶/取消置顶：`Win + Ctrl + T`



### 不常用

#### 工作区

允许用户把多个程序按预设的**窗口位置、大小和配置**保存为一个工作区，并可通过快捷方式或点击“一键恢复”这些布局状态。

看着挺有用的，特别是大屏/多屏下切换特定的工作模式，但对我的两个小屏幕使用场景来说暂时没有意义，感觉以后能用上。



#### FancyZones（窗口分区）

功能强大的窗口管理器实用工具，可帮助你将窗口排列并贴入自定义布局。

对我而言 useless，windows 原生的窗口吸附就支持我大部分所需的功能了，估计是给屏幕非常大的人用的。

如果要用，建议在 `PowerToys` 设置里打开`在取消贴靠时还原窗口的原始大小`和`贴靠窗口时禁用圆角`设置，再在`编辑布局`中关闭窗口周围的间隙，这样体验和 Windows 原生的窗口吸附会更接近一点。

顺带一题，原生的窗口吸附除了把窗口拉出屏幕边缘外，还可以按 `win + z` 唤出更多形式的布局



#### 窗口裁剪器

帮助你通过创建更小的裁剪窗口来专注于应用程序的特定部分。 可以创建实时反映更改的缩略图，或将应用程序部分提取到独立窗口中，以便更好地执行多任务和屏幕空间管理。

创建实时反映更改的缩略图感觉没什么用，将应用程序部分提取到独立窗口中在我电脑上有 bug （悲）

### 键盘管理器

键盘管理器（Keyboard Manager） 是 PowerToys 中用于**自定义键盘行为**的一个实用工具。它允许用户**重新映射单个按键、组合快捷键甚至将按键映射成文本或启动程序的动作**，从而实现更高效且按需的输入体验。

和我的 mykeymap 定位冲突了，但它好像不会被游戏检测为外挂，感觉拿来重映射按键挺好用的，比如把 `W`  和 `space` 映射成同一个键，都是跳跃。
